# 🔍 Passive Reconnaissance Demo

A cybersecurity education project demonstrating how much data a website can silently collect from visitors — without any permission prompts.

## What It Collects

- Browser & OS fingerprint (User-Agent, platform)
- Screen resolution & color depth
- Timezone & language
- CPU cores & RAM
- Network type (4G/3G/WiFi)
- Battery level
- Canvas fingerprint (unique device ID)
- Local IP via WebRTC
- GPU model via WebGL

## How It Works

All data is collected client-side using standard JavaScript APIs that browsers expose to any website. No special permissions are required.
